# Assets Folder

This folder should contain the following image files for the app:

- `icon.png` - App icon (1024x1024px recommended)
- `splash.png` - Splash screen image (1242x2436px recommended)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

## Quick Setup

You can create placeholder images or use online tools:

1. **For quick testing**, you can create simple colored squares using any image editor
2. **Recommended tool**: Use [Expo's asset generator](https://docs.expo.dev/guides/app-icons/) or any online icon generator

## Temporary Workaround

If you want to run the app without these assets immediately, you can temporarily comment out the icon and splash image references in `app.json`, but this is not recommended for production.
